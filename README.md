# Journal Du Code
Le Journal Du Code, c'est une initiative visant à créer un journal libre pour parler d'informatique, de technologies, ainsi que parler de code.
L'idée c'est que chacun puisse écrire un article, et ensuite on le met dans la prochaine édition.

# Publier un article
Pour publier un article, il existe deux méthodes.

## 1. Faire une pull request
Vous pouvez simplement faire une pull request en y ajoutant votre article.

## 2. Envoyer un E-Mail
Vous pouvez également envoyer un E-Mail à l'adresse wilrakov@proton.me avec votre article et un petit message pour m'introduire à votre sujet.
